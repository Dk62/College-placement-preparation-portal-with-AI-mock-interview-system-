# Documentation Index

This folder contains both phase-wise developer documentation and source-code documentation for the College Placement Preparation Portal with AI Mock Interview.

## Recommended Reading Order

1. `Project_Documentation_With_Source_Code.md` - complete project report with architecture, APIs, database, workflows, source-code snippets, and explanations.
2. `Phase1_Server_and_Architecture.md` - backend bootstrap and Express architecture.
3. `Phase1_Database_and_Auth.md` - database relationships and authentication engine.
4. `Phase1_Frontend_Auth_UI.md` - frontend login and registration UI.
5. `Phase2_Admin_and_TPO.md` - admin and TPO dashboards.
6. `Phase2_Student_and_Company.md` - student and company dashboards.
7. `Phase3_AI_and_Interviews.md` - AI mock interview and resume analysis.
8. `Phase4_Assessments_and_Drives.md` - aptitude tests and placement drives.
9. `Phase5_Profile_and_ATS.md` - profile, resume builder, and ATS.
10. `Phase6_Core_Views_and_Recovery.md` - password recovery and core views.
11. `Phase7_Backend_API_Integrations.md` - backend API and deployment integration notes.
